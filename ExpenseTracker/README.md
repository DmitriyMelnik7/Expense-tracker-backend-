Simple expense tracker (backend part) created using:
- .NET 10
- ASP.NET CORE Web API
- Entity Framework Core
- MS SQL Server
- FluentValidation
- Cookie-based auth
- Json Patch
- Postman
- Git

Functions of app are user's register and auth, CRUD-operations with budgets, expense/income categories, accounts management, transactions.

Quick start: 
1. Clone this repository
2. Setup database connection string in appsetings.json
3. Add migrations
4. Run app

!!!This project is still developing
